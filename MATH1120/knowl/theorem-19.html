Theorem 17.3.4. Properties of Matrix Multiplication.

Let \(A\text{,}\) \(B\) and \(C\) be appropriately sized matrices. Then

  1. \(A(BC)=(AB)C \qquad\) (Associative Law)

  2. \(A(B+C) = AB + AC \qquad\) (Left Distributive Law)

  3. \((A+B)C = AC + BC \qquad\) (Right Distributive Law)

  4. \(\displaystyle k(AB) = (kA)B = A(kB)\)

  5. \(I_mA = A = AI_n\) if \(A\) is an \(m \times n\) matrix \(\quad\) (Identity Law)

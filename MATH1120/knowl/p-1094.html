Consider the non-homogenous system of linear equations

\begin{equation} A\mathbf{x} = \mathbf{b}, \ \mathbf{b} \neq 0\tag{17.3.1} \end{equation}

and the corresponding homogeneous system

\begin{equation} A\mathbf{x} = \mathbf{0}\tag{17.3.2} \end{equation}

  1. If (17.3.1) has two distinct solutions then it has an infinite number of solutions.

  2. If (17.3.1) has a unique solution then (17.3.2) has only the trivial solution \(\mathbf{x}=\mathbf{0}\text{.}\)

  3. If (17.3.1) has an infinite number of solution then so does (17.3.2).

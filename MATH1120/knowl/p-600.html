The second method that we will look at aims to approximate the solution to the initial-value problem

\begin{equation*} y'=f(x,y), \hspace{5mm} y(x_0)=y_0 \end{equation*}

numerically (i.e to produce a table of values for the function). Given a point \((x_n,y_n)\) on the graph of the solution \(y(x)\text{,}\) Eulerâ€™s method approximates the next point \((x_{n+1},y_{n+1})\) on the graph of \(y(x)\) using the linear approximation

\begin{equation*} \Delta y=f(x_n,y_n)\Delta x \end{equation*}

Thus if \(x_{n+1}=x_n+\Delta x\text{,}\) where \(\Delta x\) is small, then

\begin{equation*} y_{n+1}=y_n+f(x_n,y_n)\Delta x \end{equation*}

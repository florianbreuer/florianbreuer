Recall that the linear differential equation

\begin{equation*} \dfrac{dx}{dt} = ax, \, \, a \in \mathbb{R}\text{,} \end{equation*}

has the solution (via separation of variables)

\begin{equation*} x(t) = C e^{at}\text{,} \end{equation*}

where \(C\) is an arbitrary constant. Consider now the system of two linear differential equations

\begin{align*} \dot{x}_1 \amp = \dfrac{dx_1}{dt} = ax_1 + bx_2\\ \dot{x}_2 \amp = \dfrac{dx_2}{dt} = cx_1 + dx_2 \end{align*}

where \(a, \, b, \, c, \, d \in \mathbb{R}\text{,}\) which can be written in matrix notation as

\begin{equation} \dot{\bm{x}} = A \bm{x}\tag{20.2.1} \end{equation}

where \(\bm{x} = \begin{pmatrix} x_1 \\ x_2 \end{pmatrix}\text{,}\) \(\dot{\bm{x}} = \begin{pmatrix} \dot{x}_1 \\ \dot{x}_2 \end{pmatrix}\) and \(A = \begin{pmatrix} a \amp b \\ c \amp d \end{pmatrix}\text{.}\) These equations are “coupled”, i.e. the derivative of \(x_1(t)\) depends on both \(x_1(t)\) and \(x_2(t)\) and likewise for the derivative of \(x_2(t)\text{.}\) Thus we can’t solve the first equation unless we can solve the second and vice versa. Note that if \(A\) is diagonal then the equations become uncoupled and we could solve each separately.
